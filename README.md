# Netflix EDA

A data analysis project exploring trends in Netflix content using a Kaggle dataset.
- `netflix_titles.csv` — dataset used for analysis
- `analyze.ipynb` — Jupyter notebook with analysis and visualizations

## Tools
- Python
- Jupyter Notebook
- Pandas / Matplotlib / Seaborn 

##  How to Use
1. Clone the repository.
2. Open `analyze.ipynb` in Jupyter Notebook.
3. Run all cells to see insights and visualizations.


## What I Did
- Data cleaning and preprocessing  
- Exploratory data visualization  
- Drawing insights and storytelling from data
